QUICK START (Copy & Paste)
════════════════════════════════════════════════════════════════════════════

1. Setup (2 minutes)
   python3.10 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   cp .env.example .env

2. Start Database (1 minute)
   docker-compose up -d
   sleep 10

3. Run Migrations (30 seconds)
   alembic upgrade head

4. Start Server (30 seconds)
   python -m uvicorn main:app --reload

5. Test (30 seconds)
   # In another terminal:
   python test_auth.py
   # Expected: Results: 7/7 tests passed ✓

6. Deploy (5 minutes)
   railway up
   # Save the URL from logs

TOTAL TIME: ~10 minutes

API ENDPOINTS (4 total)
════════════════════════════════════════════════════════════════════════════

1. POST /auth/signup
   Input:  { email, password }
   Output: { access_token, user_id }
   Status: 200 (success), 400 (email exists)

2. POST /auth/login
   Input:  { email, password }
   Output: { access_token, user_id }
   Status: 200 (success), 401 (invalid credentials)

3. GET /auth/me (Protected)
   Header: Authorization: Bearer <token>
   Output: { id, email, role, created_at }
   Status: 200 (success), 401 (invalid token)

4. GET /health
   Output: { status: "ok" }
   Status: 200 (always)
